# マルチデバイス同期チェックリスト

別のMacで作業を再開する際は、以下の項目を確認してください。

## 1. ツール・認証
- [ ] **GitHub CLI (`gh`):** ログイン済みか確認 (`gh auth status`)。
- [ ] **Git:** 最新のコミットがプルされているか、またはクラウド同期が完了しているか。

## 2. 環境
- [ ] **Node.js:** `package.json` に適合するバージョンか。
- [ ] **Dependencies:** `npm install` が実行済みか。

## 3. Gemini へのコンテキスト受け渡し
新しい環境で Gemini CLI を起動した際、まず最初に以下のコマンドを（ユーザーが）入力、または Gemini が自発的に読み取るようにしてください：
> 「.gemini/ ディレクトリ内の全てのファイルを読み込んで、現在のプロジェクト状況と君の役割を把握して」

これにより、私が「何者で」「今何をしていて」「次は何をすべきか」を一瞬で思い出せます。
